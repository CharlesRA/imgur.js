Imgur.js

Classes

  • ImgurClient

    

An unofficial powerful library for interacting with the Imgur API

Install

npm install imgur-api.js

Usage

const ImgurClient = require("imgur-api.js");
const client = new ImgurClient({
	clientId: "myClientId",
	accessToken: "userAccessToken",
});
